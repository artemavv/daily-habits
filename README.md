# Daily Tracker

A learning project to build a todo tracker in various implementations.

## Overview

This project explores different approaches to building a todo/habit tracker application. The goal is to learn and compare different technologies and frameworks by implementing the same core functionality multiple times.

## Implementations

### 1. Vanilla JavaScript + HTML

**Practice**
- Local storage for data persistence
- Dynamic DOM manipulation
- Event handling
- Responsive design with CSS

**Features:**
- Add and delete TODO items
- Timestamp tracking for checkboxes
- Visual indicators for stale (old) checked items

**Usage:**
Simply open `index.html` in a web browser.
